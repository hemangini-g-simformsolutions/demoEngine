## Installation
Add this line to your application's Gemfile:

```ruby
gem 'demo_engine', git:'https://github.com/hemangini-g-simformsolutions/demoEngine.git', branch:'main' 
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install demo_engine
```
